# text-transform

## `uppercase`

將字體改大寫

```html
<p>
  Initial String
  <strong>Lorem ipsum dolor sit amet, consectetur adipisicing elit…</strong>
</p>
<p>
  text-transform: uppercase
  <strong><span>Lorem ipsum dolor sit amet, consectetur adipisicing elit…</span></strong>
</p>
```

```css
span {
  text-transform: uppercase;
}
strong {
  float: right;
}
```
